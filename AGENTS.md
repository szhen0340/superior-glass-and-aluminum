# AGENTS.md

## Project

Single-page TanStack React Start app (SSR via Nitro). All source lives in `src/`.

## Developer Commands

```bash
bun install              # install deps
bun --bun run dev       # dev server on port 3000
bun --bun run build     # production build
bun --bun run start     # start production server
bun --bun run test      # Vitest (no test files currently)
bun --bun run lint      # ESLint
bun --bun run format    # Prettier
bun --bun run check     # prettier --write + eslint --fix
```

Use `bun --bun run` prefix for all non-dev scripts (bun runtime isolation). `dev` uses plain `bun run dev`.

## Routing and Codegen

**Do not edit `src/routeTree.gen.ts` manually.** It is auto-generated by `@tanstack/router-plugin` on every `vite dev` and `vite build`. To add or rename routes, edit files under `src/routes/` — the codegen will update the route tree automatically.

Router factory: `src/router.tsx` exports `getRouter()`.

## Path Aliases

`@/*` maps to `./src/*` (configured in `tsconfig.json` and `vite-tsconfig-paths`).

## Tailwind CSS v4

This project uses Tailwind v4, which configures via CSS (`src/styles.css`), not a JS config file. There is no `tailwind.config.js`.

## Linting / Formatting

ESLint uses `@tanstack/eslint-config` with the `tanstackConfig` preset — no local overrides.

Prettier config (`prettier.config.js`): single quotes, no semicolons, trailing commas all.

The `check` script runs Prettier first (write), then ESLint (fix). Run `check` before committing; do not skip the Prettier step.

## Gitignore Artifacts

`count.txt` and `todos.json` are gitignored local dev artifacts — safe to ignore or delete.

## Testing

No test files exist currently. Test dependencies (`@testing-library/react`, `jsdom`, `vitest`) are installed. When adding tests, Vitest v3 uses auto-detection or a `vitest.config.ts` (none currently present).

## Stack

- **Runtime**: Bun
- **Framework**: TanStack React Start (React 19 + TanStack Router + Nitro SSR)
- **Build**: Vite v7
- **Styling**: Tailwind CSS v4 (`@tailwindcss/vite` plugin)
- **Path aliases**: `vite-tsconfig-paths`
